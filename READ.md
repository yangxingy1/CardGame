# Double Peaks Solitaire (双峰纸牌)

基于 Cocos2d-x C++ 引擎开发的一款经典纸牌消除游戏。玩家需要通过巧妙的策略与计算，在一层层叠放的山峰中寻找消除的契机，清空桌面上的所有卡牌。

## 🎮 游戏玩法与规则 (How to Play)

本游戏采用类似经典 Tri-Peaks (三峰插花) 的核心消除规则，并引入了更具厚度感与挑战性的“双峰”布局。

### 核心规则
* **连击消除 (+/- 1 规则)**：玩家只能点击消除与当前**屏幕下方基准牌（废牌栈顶）**点数相差为 1 的卡牌。
* **循环相连**：卡牌点数支持首尾循环连接，即 `K` 可以消除 `A`，`A` 也可以消除 `K`。花色与颜色对消除不产生影响。

### 桌面布局
* **双山峰区 (主牌区)**：屏幕上方包含两座并排的山峰。每座山峰由 14 张牌组成，采用 `8(地基) + 3 + 2 + 1` 的特殊倒金字塔结构。地基部分拥有 4 层厚度，玩家必须从上至下、像剥洋葱一样逐层解锁被压在底下的卡牌。
* **遮挡机制**：只有完全**没有被其他卡牌压住**的牌才会被翻开并允许点击。
* **额外牌堆与消除栈 (堆牌区)**：
  * 当主牌区无牌可消时，点击左下角的**额外牌堆**可翻开一张新牌，覆盖在右下角的**消除栈**上，作为新的基准牌。
  * **撤销功能 (Undo)**：走错了一步？直接点击右下角的消除栈，即可触发时间倒流！被消除的牌会飞回山峰并恢复遮挡状态，极大地增强了游戏的容错率与策略深度。

### 胜负判定
* **🏆 胜利**：成功消除主牌区（双山峰）的所有卡牌。
* **💀 失败**：额外牌堆已被全部抽空，且主牌区没有任何一张可点击的牌能与基准牌匹配。

## 🛠️ 技术特性 (Cocos2d-x Features Utilized)

本项目深度应用了 Cocos2d-x 2D 游戏引擎的各项核心功能，构建了流畅且严谨的卡牌交互系统：

* **精灵与纹理渲染 (Sprite & Texture)**
  * 利用 `Sprite` 动态切换卡牌的正面与背面纹理。
  * 屏幕适配：根据设备可见区域 `getVisibleSize` 计算缩放系数 `setScale`，确保背景图完美铺满屏幕。
* **坐标系与渲染层级 (Coordinate System & Z-Order)**
  * 通过精密的 `Vec2` 坐标偏移量呈现卡牌的堆叠厚度感。
  * 深度运用 `LocalZOrder`，确保卡牌在移动、堆叠时的视觉前后关系（如让飞行的卡牌始终处于最上层）。
* **物理碰撞与状态算法 (BoundingBox Intersection)**
  * 抛弃死板的写死逻辑，使用 `intersectsRect` 进行 AABB 矩形碰撞检测，自动算出卡牌之间的物理遮挡关系。
* **动作系统 (Action & Animation)**
  * 结合 `MoveTo` 与 `CallFunc`，使用 `Sequence` 将平滑的位移动画与翻牌逻辑串联，提供极致流畅的视觉反馈。
* **触摸事件分发 (EventDispatcher)**
  * 注册单点触摸监听器 (`EventListenerTouchOneByOne`)。
  * 利用严格的事件优先级和范围判断 `containsPoint`，精确拦截（`SwallowTouches`）额外牌堆、消除栈与主牌区的点击事件。
* **历史栈与命令模式 (Stack-based Undo System)**
  * 结合 C++ 的 `std::stack` 数据结构，精准记录每一步操作的位移、Z轴层级及卡牌翻转状态快照，实现了完美的无缝回退功能。
